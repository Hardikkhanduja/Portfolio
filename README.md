## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/Hardikkhanduja/Portfolio.git

# Open in browser
cd Portfolio
# Open index.html or use a local server
```

**Using Live Server (VS Code):**
- Install "Live Server" extension
- Right-click `index.html` → "Open with Live Server"
